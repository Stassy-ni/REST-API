Сложение обыкновенных дробей

1. Правило сложния дробей с одинаковыми знаменателями

Чтобы сложить дроби с одинаковыми знаменателями, нужно:

  1. Сложить их числители.
  2. Знаменатель оставить тот же.
  3. При необходимости сократить дробь.

\(\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}\)

Пример: \(\frac{2}{5} + \frac{1}{5} = \frac{2+1}{5} = \frac{3}{5}\)

Задача. В первый день вскопали \(\frac{4}{15}\) всего огорода, а во второй день –\(\frac{7}{15}\) всего огорода. Какую часть огорода вскопали за \(2\) дня?

Решение: Чтобы найти, какую часть огорода вспахали за \(2\) дня, нужно сложить, сколько вспахали в первый день, и сколько вспахали во второй день:

\(\frac{4}{15} + \frac{7}{15} = \frac{11}{15}\) часть.

Ответ: За \(2\) дня вспахали \(\frac{11}{15}\) часть огорода.

2. Правило сложения дробей с разными знаменателями

Чтобы сложить дроби с разными знаменателями, надо:

  1. Найти наименьшее общее кратное (НОК) знаменателей.
  2. Привести дроби к общему знаменателю, умножив числитель и знаменатель каждой дроби на соответствующее число.
  3. Сложить числители, а знаменатель оставить тот же.
  4. При необходимости сократить дробь.

Пример: \(\frac{1}{3} + \frac{1}{4}\)

НОК(3, 4) = 12

\(\frac{1}{3} = \frac{1 \cdot 4}{3 \cdot 4} = \frac{4}{12}\)

\(\frac{1}{4} = \frac{1 \cdot 3}{4 \cdot 3} = \frac{3}{12}\)

\(\frac{4}{12} + \frac{3}{12} = \frac{4+3}{12} = \frac{7}{12}\)

Задача. Два велосипедиста двигаются навстречу друг друга. Первый велосипедист за \(1\) ч проезжает \(\frac{1}{5}\) расстояния между ними, а второй – \(\frac{1}{4}\) этого расстояния. На какую часть расстояния они сближаются каждый час?

Решение: Для решения задачи надо сложить расстояние, которое за \(1\) ч проезжает первый велосипедист \(\frac{1}{5}\) и расстояние, которое проезжает за \(1\) ч второй велосипедист \(\frac{1}{4}\).

Приведём дроби к наименьшему общему знаменателю \(20\):

\(\frac{1}{5} = \frac{4}{20}\), \(\frac{1}{4} = \frac{5}{20}\).

\(\frac{1}{5} + \frac{1}{4} = \frac{4}{20} + \frac{5}{20} = \frac{9}{20}\) – часть.

Ответ: Каждый час велосипедисты сближаются на \(\frac{9}{20}\) часть.

3. Правило сложения смешанных чисел

Чтобы найти сумму смешанных чисел, надо:

  1. Дробные части этих чисел привести к наименьшему общему знаменателю.
  2. Отдельно выполнить сложение целых и отдельно дробных частей.
  3. При необходиости сократить дробь, выделить целую часть и прибавить её к полученной целой части.

Пример: \(6 \frac{4}{9} + 2 \frac{5}{6}\)

Складываем целые части: \(6 + 2 = 8\).

Складываем дробные части: \(\frac{4}{9} + \frac{5}{6}\).

НОК(9, 6) = 18

\(\frac{4}{9} = \frac{8}{18}\), \(\frac{5}{6} = \frac{15}{18}\)

\(\frac{8}{18} + \frac{15}{18} = \frac{23}{18}=1 \frac{5}{18}\)

Итог: \(8 + 1\frac{5}{18} = 9 \frac{5}{18}\)

Задача 1. Маша собрала \(6 \frac{1}{8}\) кг малины, а Миша – \(5 \frac{6}{8}\) кг. Сколько килограммов малины собрали дети?

Решение: Для того, чтобы найти, сколько килограммов малины собрали дети, нужно сложить, сколько килограммов собрала Маша, и сколько килограммов собрал Миша:

\(6 \frac{1}{8} + 5 \frac{6}{8} = 11 \frac{7}{8}\) кг.

Ответ: \(11 \frac{7}{8}\) кг малины собрали дети.

Задача 2. Скорость теплохода по озеру равна \(26 \frac{3}{4}\) км/ч. Найдите скорость теплохода по течению реки, если известно, что скорость течения реки равна \(3 \frac{1}{12}\) км/ч.

Решение: Чтобы найти скорость теплохода по течению реки, надо к скорости теплохода по озеру \(26 \frac{3}{4}\) км/ч прибавить скорость течения реки \(3 \frac{1}{12}\) км/ч.

Приведём дроби к наименьшему общему знаменателю \(24\):

\(26 \frac{3}{4} = 26 \frac{18}{24}\), \(3 \frac{1}{12} = 3 \frac{2}{24}\).

\(26 \frac{3}{4} + 3 \frac{1}{12} = 26 \frac{18}{24} + 3 \frac{1}{12} = 29 \frac{20}{24} = 29 \frac{5}{6}\) км/ч.

Ответ: Скорость теплохода по течению реки \(29 \frac{5}{6}\) км/ч.
